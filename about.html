Je m’appelle Rakotondrabe Miora Fanomezana, j’ai 22 ans et je termine actuellement mon baccalauréat en informatique a l’UQAC. Ce que j'aime dans le développement web, c'est qu'il réunit trois de mes centres d'intérêt : la programmation, la résolution de problèmes et l'apprentissage continu. Pourquoi ? Parce que le développement web est un domaine en perpétuelle évolution, avec de nouvelles technologies, Framework et outils qui apparaissent régulièrement. En plus du développement d'applications web, je suis passionné par les mathématiques, la musique et l'intelligence artificielle. Pour moi, les mathématiques sont un langage universel qui permet de partager des idées et des connaissances pour résoudre des problèmes. Quant à l'intelligence artificielle, c'est un domaine en pleine expansion qui, ce que j'apprécie particulièrement, relie les mathématiques, l'informatique et les défis réels du monde.

Au fil de mes années à l'université et à travers mes recherches personnelles, j'ai su développer et perfectionner les compétences suivantes :

Langages de programmation :
C# : Utilisé principalement pour le développement d'API web robustes avec le framework .NET, offrant une architecture évolutive et maintenable.
JavaScript : Employé dans le développement frontend, en particulier avec React, pour créer des interfaces utilisateur dynamiques et réactives.
Python : Exploité dans le domaine de l'intelligence artificielle, notamment avec des bibliothèques telles que TensorFlow et scikit-learn pour le machine learning et la modélisation de données.
Java : Utilisé pour le développement d'applications mobiles performantes sous Android Studio.
C++ : Langage polyvalent pour des applications haute performance, utilisé pour des projets nécessitant une gestion fine des ressources.

Base de données : MySQL, Oracle SQL, SQL Server
Script SQL : Capacité à écrire des scripts SQL complexes pour manipuler et interroger efficacement des bases de données relationnelles.
scripts PL/SQL : Expérience dans la création de procédures stockées avancées avec Oracle, optimisant les performances des transactions.
ORM : Compétence pour interagir avec des bases de données à l'aide d'ORM (comme Entity Framework), garantissant un accès aux données fluide et un code plus maintenable.

Concept avancé:
Sécurité: Capable de développer des applications sécurisées en prenant compte les vulnérabilités et en mettant en oeuvre les meilleures pratiques de sécurité.
Authentification: Expérience dans la mise en place d'authentification robuste pour les applications, notamment avec la gestion de JWT.
Publication: Compétent dans l'utilisation de Docker pour conteneuriser des applications et les déployer dans le cloud. Expérience avec Kubernetes pour l'orchestration et le déploiement à grande échelle.
